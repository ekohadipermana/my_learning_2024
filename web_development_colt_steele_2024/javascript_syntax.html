Javascript Syntaxes

Collected from Colt Steele and Angela course

Basic Primitive Types: Number, String, Boolean, Null, Indefined

syntax: // -- create comment

syntax: * / + - % -- multiply, divide, add, subtract, modulo

NaN: Not a Number

syntax: let someName = value -- to create variable

syntax: score += 5 -- add 5 to the score itself.

syntax: score++ -- increment by one

syntax: const thePi = 99 -- to declare constant

use let instead of var (var is outdated but still can be used)

Boolean is true or false (lower case)

Naming convention is just like Python!

About string. You can use double or single quotes.

Strings are indexed, start from 0.

Syntax: stringVar[0] -- will give you the character on position 0

Syntax: stringVar.length -- give the length of the string

Concatenation: "River" + " " + "Phoenix"

String Method

syntax: stringVar.toUpperCase(); -- make it upper case

syntax: stringVar.toLowerCase(); -- make it lower case

syntax: stringVar.trim(); -- to eliminate empty spaces in the beginning or the end

you can chain the method together like this:

stringVar.toUpperCase().toLowerCase().trim()

IndexOf

let tvShow = 'catdog';
tvShow.indexOf('cat'); --- result 0
tvShow.indexOf('z'); --- result -1 (not found)

Slice

let str = 'supercamp';
str.slice(0,5); -- result: 'super'
str.slice(5); // result: 'camp'

Replace

let words = 'super camp';
words.replace('super', 'big'); // 'big camp'

Repeat

let varName = 'lol';
varName.repeat(5); // 'lollollollollol'

String Template Literals

Syntax: `I counted ${3 + 4} sheep`; // 'I counted 7 sheep'

So you dont need to add plus sign etc. Super useful.

You must use back-tick ` ` above tab key

Undefined and Null

let loggedInUser = null; // value is nothing
When a user signed in:
loggedInUser = 'Alan';

Math Object

Syntax: Math.PI; // 3.141592

Math.round(4.9); // 5

Math.abs(-456); // 456

Math.pow(2,5); // 2 power 5

Math.floor(3.9999); // removes decimal (3)

Math.random(); // gives random between 0.0 to 0.99999

To get random number between 1 and 10:

Math.floor(Math.random() * 10) + 1;

Decision Making in Javascript

Comparison: > < >= <= == != === !==

Double equality: doesnt care about type

Triple equality: cares about type

Console, Alert and Prompt

Run the Javascript with file

create js file, connect it with a html via script tag, run it in a browser and open the browser console.

Conditionals

If statement

let rating = 3;
if (rating === 3){
console.log('Supercamp');
}

Else If

let rating = 3;
if (rating === 3){
console.log('Supercamp');
}
else if (rating === 2) {
console.log('OK');
}

Else: for last thing, if everything doesnt work, use else.

You can nest conditionals inside conditionals

Logical Operators

&& (and)
|| (or)
! (not)

Arrays

let students = ['eko', 'hadi', 'permana'];

Array Random Access

students[0]; // 'eko'

Modifying Array

students[0] = 'kasep'; // changing 'eko' to 'kasep'

Array methods

Push Pop shift unshift

students.push('dihya'); // add 'dihya' to the end.

students.pop(); // remove last element

students.shift(); // remove the first element

students.unshift('kylo'); // adding kylo to the start

const array3 = array1.concat(array2); // to join two strings

array1.includes('blue'); // to check is 'blue' in array1

array1.indexOf('bison'); to check the index number of 'byson'

array1.reverse(); // to reverse the order of elements

array1.slice(3); // copy the elements start from 3 to the end

array1.slice(2, 4); // copy from 2 to before 4

array1.splice(1, 0, 'Feb'); // insert at index 1

array1.splice(4, 1, 'May'); // replaces 1 element at index 4

array1.sort(); // to sort alphabetically

array1.toString(); // convert the elements into one string
